## Preview

```tsx
import { AdwActionRow, AdwPreferencesGroup } from "@gtk-js/adwaita";

<AdwPreferencesGroup title="Account">
  <AdwActionRow
    title="Username"
    subtitle="hwittenborn"
  />
  <AdwActionRow
    title="Email"
    subtitle="hunter@example.com"
    activatable
    onActivated={() => alert("Edit email")}
  />
</AdwPreferencesGroup>
```

## Prefix & Suffix Slots

Use `prefix` and `suffix` props (or pass children which go into suffixes):

```tsx
<AdwActionRow
  title="Volume"
  prefix={<GtkImage iconName="audio-volume-high-symbolic" />}
  suffix={<GtkScale value={75} min={0} max={100} />}
/>
```

## Style Classes

| Class | Effect |
|-------|--------|
| `.property` | Inverts emphasis: subtitle becomes prominent, title becomes dimmed |
| `.monospace` | Applies monospace font to subtitle only |
